
# Experiment 2 Web Applications

## Description
This project consists of three mini web applications developed as part of Experiment 2. 
The applications demonstrate how HTML, CSS, and JavaScript can be used to build interactive 
and dynamic user interfaces.

## Experiment 2.1 Live Character Counter
A real-time word counter application that updates as the user types.

## Experiment 2.2 Dynamic Product Filter
Filters products based on selected category.

## Experiment 2.3 SVG Drawing Tool
Allows users to draw on an SVG canvas using mouse events.

## Learning Outcomes
- Learned HTML page structure  
- Applied CSS styling  
- Used JavaScript events  
- Implemented DOM manipulation  
- Built interactive web pages
